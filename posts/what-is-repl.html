What is REPL?

REPL stands for Read Eval Print Loop. It is a interactive software that takes an argument as input, exaluate the input, executes an action, and return the result of the action to the user. The command line shell of operating systems is an example of a REPL. Other example is the REPL for some programming languages such as Python and Lisp.

To access the Python REPL for example, first you must have Python installed in your computer (some operating systems come with Python installed already. Some do not come with it install. Here is the link to download it and you may find information to install it there too). Then you open the command line terminal of your computer (that is a REPL by itself) and type python in lower case, and boom. The Python REPL will open. That easy. Then type math equations like 1 + 2 and see that it makes the calculation and prints the result right below your input.

REPL is not like a person that we can talk and say whatever we want, it has its own special language that we have to learn in order to use the it. Also, this language may vary depending of which REPL you are using. If you are using the Python one, then it will only accept commands supported by the Python language. The same is applied for other REPLs.
